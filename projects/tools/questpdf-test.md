# TestCodeSpace-QuestPDF - PDF 文档生成与编辑测试

## 核心需求
探索和测试 QuestPDF 库在 .NET 应用程序中生成和编辑 PDF 文档的能力，主要包括：
1.  **文档布局**：使用 QuestPDF 的 Fluent API 设计复杂的文档布局，如发票、报告。
2.  **数据绑定**：将 C# 对象数据绑定到 PDF 模板中，实现动态内容生成。
3.  **元素支持**：测试 QuestPDF 对文本、图片、表格、条形码等各种 PDF 元素的支持。
4.  **性能优化**：评估 QuestPDF 在生成大量页面或复杂文档时的性能。

## 开发目标
-   掌握 QuestPDF 库的使用，为未来项目中的 PDF 报告功能提供技术储备。
-   创建可复用的 PDF 模板和组件。

## 建议技术栈
-   **语言**: C#
-   **框架**: .NET 8 (或更高版本)
-   **库**: QuestPDF
-   **UI**: Console Application (用于测试生成) 或 WPF/WinForms (用于集成预览)

## Solo 模式开发提示
-   **指令示例**: "使用 QuestPDF 库，生成一份包含表格和图片的简单发票 PDF 文档。"
-   **关注点**: 优先实现核心的 PDF 布局和内容填充，复杂交互功能可暂时忽略。
