# FileControlManager - 文件控制管理工具

## 核心需求
开发一个桌面应用程序，用于高效地管理文件和文件夹，提供以下功能：
1.  **文件/文件夹操作**：复制、移动、删除、重命名文件和文件夹。
2.  **权限控制**：可选地集成用户权限管理，限制对特定文件或目录的操作。
3.  **监控与日志**：记录文件操作日志，便于追踪和审计。
4.  **搜索与过滤**：根据文件名、类型、大小等属性快速搜索和过滤文件。

## 开发目标
-   提供直观的用户界面，简化文件管理任务。
-   确保文件操作的稳定性和安全性。
-   支持大批量文件操作的性能优化。

## 建议技术栈
-   **语言**: C#
-   **框架**: .NET 8 (或更高版本)
-   **UI**: WinForms 或 WPF (推荐 WPF 以提供更现代的 UI 和更好的扩展性)
-   **数据存储**: 用于存储配置和日志的轻量级数据库 (如 SQLite) 或文件系统。

## Solo 模式开发提示
-   **指令示例**: "创建一个 WPF 应用程序，实现文件复制和移动功能，并显示操作进度。"
-   **关注点**: 优先实现核心的文件操作逻辑，权限和监控功能可作为后续迭代。
