# Useful-Application - 实用应用程序集合

## 核心需求
开发一系列小型、独立的实用应用程序，解决日常工作或生活中的特定问题，例如：
1.  **记账本 (Cashbook)**：记录日常收支，进行简单的财务统计。
2.  **自动抢票工具 (AutoTick)**：监控票务信息，实现自动化抢票功能。

## 开发目标
-   提供简洁、高效的解决方案。
-   易于使用，无需复杂配置。
-   每个应用功能单一，专注于解决特定痛点。

## 建议技术栈
-   **语言**: C#
-   **框架**: .NET 8 (或更高版本)
-   **UI**: WinForms (快速开发，轻量级) 或 WPF (提供更好的用户体验)
-   **数据存储**: 记账本可使用 SQLite 或文件存储；抢票工具可能需要内存存储或临时文件。
-   **网络**: 抢票工具需要 HTTP 请求和解析网页内容。

## Solo 模式开发提示
-   **指令示例**: "创建一个 WinForms 记账本应用，实现收支记录和简单的统计功能。"
-   **关注点**: 优先实现核心业务逻辑，界面设计可保持简洁实用。
