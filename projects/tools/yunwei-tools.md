# YunWeiTools - 运维工具集

## 核心需求
开发一个综合性的运维工具集，包含多个独立或关联的小工具，以提高运维效率，例如：
1.  **网络监控工具**：实时监控服务器网络状态、端口连通性。
2.  **日志管理工具**：收集、分析和展示服务器日志。
3.  **自动化部署辅助**：与 Jenkins、GitLab 等 CI/CD 工具集成，简化部署流程。
4.  **钉钉机器人集成**：发送告警、通知等信息。
5.  **文件位置系统**：管理和查找服务器上的文件。
6.  **Excel 数据处理**：读取、写入和处理 Excel 数据，用于报表生成或数据分析。

## 开发目标
-   将分散的运维任务集成到一个平台。
-   提高故障排查和日常维护的效率。
-   支持模块化，方便功能扩展。

## 建议技术栈
-   **语言**: C#
-   **框架**: .NET 8 (或更高版本)
-   **UI**: WinForms 或 WPF (根据具体工具的复杂性选择)
-   **集成**: Jenkins API, GitLab API, DingTalk API
-   **数据存储**: 配置文件、SQLite 或其他轻量级数据库。

## Solo 模式开发提示
-   **指令示例**: "创建一个 WinForms 工具，实现网络 Ping 功能，并能将结果发送到钉钉群。"
-   **关注点**: 优先实现单个工具的核心功能，然后逐步集成和完善。
