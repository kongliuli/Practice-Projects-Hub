# uDatabaseTool - 数据库工具

## 核心需求
开发一个通用的数据库管理工具，提供以下功能：
1.  **多数据库连接**：支持连接多种数据库（如 SQL Server, MySQL, SQLite）。
2.  **数据查询与编辑**：提供 SQL 查询界面，并能直观地查看和编辑表数据。
3.  **数据导入导出**：支持将数据导入导出为常见格式（如 CSV, Excel）。
4.  **数据库结构管理**：查看表结构、索引、视图等，并支持简单的结构修改。
5.  **配置管理**：管理数据库连接信息和用户配置。

## 开发目标
-   提供一个轻量级、易用的数据库管理客户端。
-   简化日常数据库操作，提高开发和运维效率。
-   支持 MVVM 模式，实现 UI 与逻辑分离。

## 建议技术栈
-   **语言**: C#
-   **框架**: .NET 8 (或更高版本)
-   **UI**: WPF (MVVM 模式，利用数据绑定和命令)
-   **数据库连接**: ADO.NET 或特定数据库的 ORM (如 MySqlConnector, Microsoft.Data.SqlClient)
-   **配置管理**: JSON 或 XML 文件存储配置。

## Solo 模式开发提示
-   **指令示例**: "创建一个 WPF MVVM 数据库工具，实现连接 MySQL 数据库并显示指定表的数据。"
-   **关注点**: 优先实现数据库连接和数据展示功能，高级功能可逐步添加。
