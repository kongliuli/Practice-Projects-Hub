# SuffixChange Tool - 文件后缀名批量修改工具

## 核心需求
开发一个桌面工具，能够实现文件的批量后缀名修改功能，主要包括：
1.  **文件选择**：用户可以方便地选择单个文件或整个文件夹进行操作。
2.  **预览功能**：在执行修改前，显示文件原名和修改后的预览名。
3.  **批量修改**：支持将选定文件的后缀名统一修改为指定后缀。
4.  **图片格式转换**：特别支持将 WebP 图片批量转换为 GIF 格式（根据原项目代码推断）。

## 开发目标
-   提供简单直观的用户界面。
-   高效稳定地处理大量文件。
-   支持常见的图片格式转换。

## 建议技术栈
-   **语言**: C#
-   **框架**: .NET 8 (或更高版本)
-   **UI**: WinForms (快速开发) 或 WPF (提供更现代的 UI)
-   **图片处理**: 使用 `Imazen.WebP` 等库进行 WebP 到 GIF 的转换。

## Solo 模式开发提示
-   **指令示例**: "创建一个 WinForms 工具，实现文件后缀名批量修改功能，并集成 WebP 到 GIF 的转换。"
-   **关注点**: 优先实现文件选择和批量修改逻辑，图片转换功能可作为独立模块实现。
