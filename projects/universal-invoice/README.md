# UniversalInvoice - 跨平台通用票据处理工具

## 核心需求
开发一个通用的票据（发票、收据等）处理工具，能够：
1.  **票据信息录入**：支持手动录入或从图片/PDF 中识别票据关键信息。
2.  **商品/服务定价**：根据商品或服务信息自动计算价格。
3.  **票据生成与打印**：根据录入信息生成标准格式的票据，并支持打印。
4.  **数据存储**：存储已处理的票据信息，方便查询和统计。

## 开发目标
-   提供灵活的票据模板，适应不同业务场景。
-   简化票据处理流程，提高效率。
-   支持跨平台部署。

## 建议技术栈
-   **语言**: C#
-   **框架**: .NET 8 (或更高版本)
-   **UI**: MAUI (实现跨平台) 或 WPF (桌面端)
-   **数据存储**: SqlSugar 或轻量级数据库 (如 SQLite)
-   **OCR**: 可集成第三方 OCR 服务进行票据识别。

## Solo 模式开发提示
-   **指令示例**: "使用 MAUI 实现 UniversalInvoice 的票据信息录入界面，包含商品名称、数量、单价等字段，并能自动计算总价。"
-   **关注点**: 优先实现核心的票据数据模型和计算逻辑，OCR 功能可作为后续扩展。
