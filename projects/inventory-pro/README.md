# InventoryPro - 企业级库存管理系统

## 核心需求
构建一个企业级的库存管理系统，支持以下核心功能：
1.  **商品信息管理**：录入、查询、修改商品的基本信息（编码、名称、描述等）。
2.  **库存变动记录**：记录商品的入库、出库、盘点等库存变动操作，并追踪操作人员。
3.  **用户权限管理**：支持多用户登录，并根据用户角色（如管理员、仓管员）分配不同的操作权限。
4.  **库存预警**：设置库存上下限，当库存量低于下限或高于上限时触发预警。
5.  **数据报表**：生成库存汇总、出入库明细等报表。

## 开发目标
-   提供稳定、高效的库存管理解决方案。
-   界面友好，操作便捷，提升仓储管理效率。
-   支持多用户并发操作。

## 建议技术栈
-   **语言**: C#
-   **框架**: .NET 8 (或更高版本)
-   **UI**: WPF (MVVM 模式，使用 `MaterialDesignInXaml` 或 `HandyControl` 提升用户体验)
-   **数据库**: **SqlSugar** (优先) 或 Entity Framework Core
-   **架构**: 采用多层架构 (Interface, Models, Service, WPF)

## Solo 模式开发提示
-   **指令示例**: "基于 SqlSugar 和 WPF MVVM，实现 InventoryPro 的商品信息管理模块，包含增删改查功能。"
-   **关注点**: 优先实现数据模型和数据库操作，UI 界面可逐步完善。
