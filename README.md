# Practice Projects Hub (实践项目中心)

## 概述
本仓库是 `moyu` 仓库的延续与升华。它不再仅仅是代码的堆砌，而是经过深度整理、需求明确的实践项目集合。

## 核心目标
- **需求驱动**: 每个项目都拥有清晰的需求文档和开发目标。
- **精选重构**: 从旧项目中筛选有意义的逻辑，使用现代技术栈（如 .NET 8, WPF MVVM, SqlSugar）进行重构。
- **Solo 模式友好**: 结构化的需求描述旨在完美适配 Trae Solo 模式，实现快速原型开发。

## 项目索引
- `projects/laygame/`: 自动化卡牌游戏逻辑与模拟。
- `projects/inventory-pro/`: 企业级库存管理系统。
- `projects/universal-invoice/`: 跨平台通用票据处理工具。
- `projects/tools/`: 实用小工具集合（如文件后缀转换等）。
- `projects/code-first-test/`: 数据库 Code First 模式测试。
- `projects/django-web-project/`: Django Web 后端测试。
